Privacy Policy

MrsMillennial Designs — Last updated: February 23, 2026

Overview

MrsMillennial Designs operates an Etsy shop at mrsmillennial.etsy.com. We use a personal desktop application called Axle to help manage our shop, including creating Pinterest pins, optimizing listings, and tracking performance. This privacy policy describes how we handle data.

Data We Collect

When the shop owner authenticates with Etsy or Pinterest, the application accesses:

  • Etsy: Shop profile, listings, listing statistics, orders, and revenue data for our shop (MrsMillennialDesigns).
  • Pinterest: User profile, boards, and pins for our connected Pinterest account.

No data is collected from customers, visitors, or any third parties. Only the shop owner's own account data is accessed.

How Data Is Stored

  • All data is stored locally on the shop owner's personal computer in a SQLite database.
  • There are no cloud servers, no external databases, and no third-party analytics.
  • OAuth tokens are stored locally and are never transmitted to any server other than the official Etsy and Pinterest API endpoints.

How Data Is Used

Data from Etsy and Pinterest is used exclusively to:

  • Display shop analytics and listing performance in a local dashboard.
  • Plan SEO improvements and manage listings.
  • Create and manage Pinterest pins to promote shop listings.
  • Track budgets and spending across shop operations.

Data is never sold, shared, or disclosed to any third party.

Third-Party Services

Our shop management tool connects to the following APIs solely for business operations:

  • Etsy Open API v3 — to read and manage shop data.
  • Pinterest API v5 — to create pins and manage boards.
  • Anthropic (Claude) API — to power an AI assistant for shop management. Anthropic's privacy policy applies.
  • OpenAI API — used as a fallback AI provider. OpenAI's privacy policy applies.

Data Retention

Data persists locally as long as the application is installed. Removing the application and its data folder deletes all stored information. There is no remote backup or cloud sync.

Data Sharing

MrsMillennial Designs does not:

  • Share data with advertisers or data brokers.
  • Collect data from users other than the shop owner.
  • Use tracking pixels, cookies, or analytics services on this website.
  • Transmit data to any server other than the official API endpoints listed above.

Security

  • OAuth 2.0 with PKCE is used for Etsy authentication.
  • OAuth 2.0 is used for Pinterest authentication.
  • All API communication uses HTTPS/TLS encryption.

Children's Privacy

This application is not designed for or directed at children under 13. No data is knowingly collected from children.

Changes to This Policy

This policy may be updated as the application evolves. The "Last updated" date at the top reflects the most recent revision.

Contact

For questions about this privacy policy, contact us via our Etsy shop: MrsMillennialDesigns on Etsy.
