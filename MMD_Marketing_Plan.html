Ad Creative Guidelines

What Works (based on Mother's Day + Classroom ads)

  • Animated "video" ads built in HTML/CSS (the play-button style we built)
  • Show the template → show the completed handprint result (before/after)
  • Emotional hook: "A memory that lasts forever" / "Capture tiny hands"
  • Social proof: "777 five-star reviews"
  • Clear value prop: "Instant download, print at home"
  • Price comparison works: "Save $2 vs Etsy" or just show the low price

Pinterest Best Practices

  • Vertical pins (2:3 ratio) perform best
  • Show the finished product (completed handprint craft, not just the template)
  • Text overlay: short, bold, readable on mobile
  • Link directly to the sale/bundle landing page, not the homepage
  • Use rich pins (already have pinterest-rich-pin meta tags)

Facebook Best Practices

  • Square or 4:5 ratio for feed ads
  • Lead with the emotional moment (kid's hands on paper)
  • Carousel ads work great for bundles (show each design)
  • Target: Women 25-45, parents, interests in crafts/DIY/teachers
  • Retarget site visitors who didn't buy

Stripe

Prices are immutable in Stripe. To change a price: create new Price object, create new Payment Link, update product HTML with new URL.

IMPORTANT: Rotate the Stripe secret key after sharing it. Go to dashboard.stripe.com/apikeys.

To Create a New Bundle

1. Pick products from stripe-links.json
2. Sum individual prices = bundle value
3. Set bundle price at 40-50% off value
4. Create Stripe product + price + payment link for the bundle
5. Build bundle detail page in /bundles/
6. Add card to bundles.html
7. If running a sale: build sale landing page with animated ad
8. Push to deploy
